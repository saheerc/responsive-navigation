# Responsive Navigation with Hamburger Menu

A fully responsive navigation bar with dropdown menus, smooth hamburger animation, and mobile-first design. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks or libraries.

## Features

- Sticky navbar with logo and links
- Dropdown menus on hover (desktop)
- Hamburger menu with animated open/close (mobile)
- Dropdown menus expand on tap (mobile)
- Active link state tracking
- Fully keyboard accessible
- No dependencies — pure HTML, CSS, JS

## Tech Stack

- HTML5 (semantic, ARIA-friendly markup)
- CSS3 (Flexbox, transitions, media queries)
- Vanilla JavaScript

## Getting Started

```bash
git clone https://github.com/saheerc/responsive-navigation.git
cd responsive-navigation
open index.html
```

## How to Use

Resize the browser window below 768px to see the mobile hamburger menu. On desktop, hover over "Products" or "Developers" to see the dropdown menus.
